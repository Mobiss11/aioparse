# Скраппер сслыок
- Парсинг контента с ссылок

## Технологии
1. `requests`
2. `beautifulsoup4`
3. `aiohttp`
4. `asyncio`
5. `aiofiles`

## Запуск

1. Установить зависимости из файла `requirements.txt` командой 
```angular2html
pip install -r requirements.txt
```
2. Запустить файл main.py командой
```angular2html
python3 main.py
```
